## Overview

Addon modules for ox_inventory adding item carrying animations, drag-and-drop crafting, and back-slot item visuals. Pulsar Framework fork of [sleepless_inventory_addons](https://github.com/Sleepless-Development/sleepless_inventory_addons) by Sleepless Development.

## Dependencies

- [ox_inventory](https://github.com/PulsarFW/ox_inventory)
- [ox_lib](https://github.com/PulsarFW/ox_lib)
