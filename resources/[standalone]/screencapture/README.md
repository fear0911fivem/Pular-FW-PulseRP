# SCREENCAPTURE
**Screenshot-basic replacement with server-side capture and remote upload for Pulsar Framework**

## Overview

Drop-in replacement for `screenshot-basic` with server-side exports for capturing player screens and uploading to remote endpoints. Provides `serverCapture` and `remoteUpload` exports and is backwards-compatible with the `requestScreenshotUpload` client export.

## Dependencies

None — fully standalone.
