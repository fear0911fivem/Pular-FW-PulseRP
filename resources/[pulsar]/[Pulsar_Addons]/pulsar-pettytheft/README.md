# pulsar-pettytheft
Petty Theft Robbery for Pulsar FW

Comes with the below fully configured 

- Mail Box Robbery
- Porch Pirate
- Parking Meter Robbery

# Configuration
Navigate to pulsar-robbery/client and create a folder named pettytheft.
Place cl_pettytheft.lua inside the new pettytheft folder.
Navigate to pulsar-robbery/server and create a folder named pettytheft.
Place sv_pettytheft.lua inside the new pettytheft folder.
Restart your server.

That’s it — the petty theft system should be fully functional.
